CSS     cascade     style sheet

definire l'aspetto degli elementi HTML


Come possiamo applicare il CSS su uno o più elementi HTML?
- inline -> attributo style
- embedded -> elemento style
- file esterno


Sintassi del CSS:


selettore {
    proprietà1: valore1;
    proprietà2: valore2;
    proprietà3: valore3;
}


Selettori CSS (specificità dei selettori - ordine di importanza crescente):
1) universal selector *
2) tag selector
3) class selector    .nomeDellaClasse
4) id selector       #Id
5) attributo style nell'HTML




Un valore per il colore può essere specificato almeno i 3 modi diversi:
- named color
- funzioni css  rgb() oppure rgba()
- hex color



font-weight:
- 100 lighter
- 300 light
- 400 normal
- 700 bold
- 900 bolder
